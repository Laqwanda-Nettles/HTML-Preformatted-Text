Preformatted Text

The pre element represents a preformatted block of text. It preserves text spaces, line breaks, tabs, and other formatting characters ignored by web browsers. It presents text exactly the way its written in the HTML file. The pre tag can be used for code snippets, quotes, and poems, so basically things you would want to keep a certain format for.
